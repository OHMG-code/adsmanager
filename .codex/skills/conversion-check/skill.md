# conversion-check

## Purpose
Sprawdź czy użytkownik faktycznie kończy kluczową akcję.

## Instructions
1. Zidentyfikuj akcję (np. dodanie klienta).
2. Przejdź cały flow.
3. Oceń:
   - gdzie użytkownik może się zatrzymać,
   - gdzie może się pomylić,
   - gdzie traci czas.
4. Wskaż:
   - punkty porzucenia,
   - zbędne kroki,
   - niejasności.

## Output
- mapa flow,
- punkty problemowe,
- konkretne usprawnienia.
