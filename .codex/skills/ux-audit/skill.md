# ux-audit

## Purpose
Przeprowadź audyt UX dla wskazanego widoku lub przepływu użytkownika.

## Instructions
1. Zidentyfikuj cel użytkownika (np. dodanie klienta, zapis kampanii).
2. Przejdź przez flow krok po kroku.
3. Oceń:
   - czy użytkownik wie co zrobić,
   - czy formularz jest czytelny,
   - czy są zbędne pola,
   - czy błędy są zrozumiałe,
   - czy są momenty frustracji.
4. Wskaż:
   - problemy UX,
   - miejsca ryzyka porzucenia akcji,
   - nieintuicyjne elementy.
5. Zaproponuj konkretne poprawki (nie ogólniki).

## Output
- lista problemów UX,
- konkretne rekomendacje zmian,
- priorytet: HIGH / MEDIUM / LOW.
