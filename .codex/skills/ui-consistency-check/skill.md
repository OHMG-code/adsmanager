# ui-consistency-check

## Purpose
Sprawdź spójność UI w całej aplikacji.

## Instructions
1. Porównaj:
   - formularze,
   - przyciski,
   - nazewnictwo,
   - komunikaty.
2. Wykryj:
   - różne nazwy dla tego samego,
   - różne style działania,
   - niespójne flow.
3. Zaproponuj ujednolicenie.

## Output
- lista niespójności,
- propozycje ujednolicenia,
- wpływ na UX.
