# Skill: UX Copy

## Cel
Tworzenie etykiet, nazw i komunikatów:
- krótkich,
- zrozumiałych,
- operacyjnych,
- bez marketingowego języka.

## Zasady

1. Używaj języka użytkownika, nie technicznego.
2. Preferuj czasowniki i działania:
   - "Dodaj klienta"
   - "Wyślij ofertę"
3. Unikaj:
   - "zarządzanie", "moduł", "system"
4. Maksymalna długość etykiety: 2–3 słowa.

## Przykłady

Złe:
- „Zarządzanie procesem kampanii”

Dobre:
- „Kampanie”

Złe:
- „Moduł komunikacji”

Dobre:
- „Skrzynka”

## Komunikaty

- mają być konkretne:
  ❌ „Błąd operacji”
  ✅ „Nie masz uprawnień do tej sekcji”

## Red flags

- zdania wielokrotnie złożone
- marketingowe slogany
- anglicyzmy bez potrzeby

## Output

- lista poprawionych etykiet
- propozycje nazw sekcji
- komunikaty UI (jeśli dotyczy)
