# final-report

## Purpose
Przygotuj końcowy raport techniczny po zakończeniu pracy.

## Instructions
1. Zbierz wszystkie wykonane zmiany.
2. Wypisz zmodyfikowane pliki.
3. Zestaw uruchomione komendy.
4. Podsumuj wyniki testów.
5. Wskaż nierozwiązane problemy lub ryzyka.
6. Zaproponuj 3 najważniejsze kolejne kroki.

## Output
Zwróć raport w sekcjach:
1. Wykonane zmiany
2. Zmienione pliki
3. Uruchomione komendy
4. Wyniki testów
5. Otwarte ryzyka
6. Następne kroki
