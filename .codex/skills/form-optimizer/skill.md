# form-optimizer

## Purpose
Popraw formularz pod kątem UX, walidacji i konwersji.

## Instructions
1. Przeanalizuj formularz:
   - kolejność pól,
   - nazwy,
   - wymagania,
   - walidację.
2. Sprawdź:
   - czy można usunąć pola,
   - czy można uprościć input,
   - czy walidacja jest czytelna,
   - czy są dobre komunikaty błędów.
3. Zaproponuj:
   - uproszczenie flow,
   - poprawę etykiet,
   - lepsze komunikaty błędów.

## Output
- lista zmian,
- poprawiona wersja formularza (opis),
- wpływ na UX (dlaczego lepiej).
